# ABMJewelry Shop - Android App
**CSS 403 | Final Project | Takhmina Ulukmanova | National Louis University**

## About the App
ABMJewelry Shop is a mobile shopping application built with Jetpack Compose and Android Studio. Users can browse jewelry collections including rings, necklaces, and earrings, view product details, and navigate between screens.

## Features
- Home screen with featured collections
- Jewelry browsing with category filters (All, Rings, Necklaces, Earrings)
- Product detail screen with size options and Add to Cart button
- Collections, Store Locator, and About ABM screens
- Smooth navigation with Back button support
- Real jewelry product images

## Technology Used
- Kotlin
- Jetpack Compose
- Navigation Component
- Material Design
- Android Studio

## Testing
- 3 automated UI tests passing successfully
- Tested on Pixel 10 emulator (API 37)
